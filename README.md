# todo-list
# Project : TODO List Application

# 기능

Todo List 기능

[시작 전]

- TODO 추가하기
- 시작 전 리스트 확인
- 체크박스 선택 시 진행 중으로 이동

[진행 중]

- 진행 중 리스트 확인
- 체크박스 선택 시 완료로 이동

[완료]

- 완료 리스트 확인
- 휴지통 선택 시 삭제
